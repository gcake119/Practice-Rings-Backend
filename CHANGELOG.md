# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2025-12-04

### Added
- 補充 Phase 2 規格：描述前端 Doraemon 主題 UI、toast 通知與粉圓體字型需求。
- 新增「手動輸入時間」流程說明，包括時間格式、驗證規則與錯誤處理策略。
- 定義「三項目總時數以 6 小時為 100%」的計算方式，作為前端進度條與勵志文案的依據。
- 描述歷史紀錄卡片的週視圖與月視圖需求（最近七天卡片、月曆 Grid、週一為一週起點）。

### Changed
- 調整前後端介面文件，將原本零散的需求整理為 Phase 2 單一規格文件。
